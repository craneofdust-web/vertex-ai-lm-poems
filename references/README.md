# References Folder

This folder stores historical/reference materials that should not mix with active runtime outputs.

## Subfolders

- `v0.1_notebookLM/`: notebookLM-era reference notes and drafts (tracked).
- `v0.2_api_results/runs/`: API-era output snapshots for comparison (local-only, gitignored by default).
- `legacy_handoff/`: previous-window handoff docs for traceability (local-only, gitignored by default).

## Policy

- Keep this folder read-mostly.
- New active runs should go to `runtime_workspaces/`.
- Deletion candidates should go to `recycle_bin/` first.
